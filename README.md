# MinionLib

用于 Slay the Spire 2 的随从（Minion）库模组。

## 支持内容

- 创建并召唤随从
- 随从主动行动（点击随从触发！）
- 强化随从的卡牌交互
- 绑定随从的卡牌交互
- Guardian 守护机制（替玩家承伤）
- 自定义目标系统（可扩展目标类型）
- 随从位置系统与移动支持
